Aeon

Your ultimate Discord bot with many utilities and API integrations.

Features

Easy to Use

Aeon works right out of the box and does not need any configuration.

Efficient

Aeon utilizes Discord's embed with simplicity in mind, used only where it is needed.

Quick

Aeon is blazing fast when it comes to responding to commands.

Featured Categories

Utility

Utilities such as Google (Assistant), Translator, Dictionary, and more.

Image Generation

Variety of image generation commands to pick from.

Fun

Tons of fun commands such as ASCII and Emojify.

Development

Development commands such as DNS, IP Lookup, Hastebin, and WHOIS.

Featured Commands

Spotify

You can get information on songs, albums, artists, users, playlists and member presences.
This command can generate cards for songs (including member presences) and albums.

osu!

Fetch user statistics, recent/top/first place scores, and get ranking leaderboard.

Also generates cards for users and supports member presences.

AniList

Search for anime, manga, characters, studios, and more from AniList. MyAnimeList command is also included.

Steam

View user information and their games and search for games from Steam.
